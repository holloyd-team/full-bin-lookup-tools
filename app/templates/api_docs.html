{{ site_name }} API

Base URL: {{ domain }}

GET /api/bin/<bin>

Retrieve BIN record by its first six digits.

Example Request

curl {{ domain }}/api/bin/411810

Example Response

{
  "bin": "411810",
  "company": "Example Bank",
  "country": "USA"
}

POST /api/report/<bin>

Submit corrections for a BIN without authentication.

Example Request

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"country": "USA"}' \
  {{ domain }}/api/report/411810

Example Response

{
  "bin": "411810",
  "country": "USA"
}
